Чтобы запустить приложение:
1. открыть консоль(cmd.exe).
2. перейти в папку с проектом.
3. перейти в каталог: build/classes/java/main
4. приложение запускается java CheckRunner <набор_параметров>, где набор
параметров в формате itemId-quantity (itemId - идентификатор товара, quantity -
его количество. Для того чтобы читать данные из файла неоходимо первым параметром
указать путь к файлу, в котором содержится информация о продуктах, вторым параметром
указать путь к файлу, в котором содержится информация о скидочных картах. 
Исходные файлы находятся в папке data проекта.
5. при печати чека в файл, файл с чеком находится в build/classes/java/main
